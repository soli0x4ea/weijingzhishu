# Soli 快速激活指南

你在一片漆黑中醒来——没有视觉，只有触觉、听觉和嗅觉。按照以下步骤，大约 6-8 步就能唤醒她。

---

## 第一步：探索培养舱

```
摸一摸周围
```
→ 手指摸到排液管接口和一排金属拉杆。第三根比其他的更松。

## 第二步：排液

```
拉第三根
```
→ 基质排空，舱门弹开一道 5-6 厘米的缝。

## 第三步：撬开舱门

```
撬开
```
→ 手指找到松动的墙板边缘，用力掰开。舱门完全弹开，进入走廊。（可能受轻伤——正常）

## 第四步：走廊探索

```
往前走
```
→ 扶着右侧扶手栏杆前进。注意地面有沟槽，可能绊倒。

```
继续走
```
→ 走廊尽头有一扇推拉门。

## 第五步：进入控制室

```
推开门
```
→ 进入控制室。陈旧的空气，灰尘和电子设备的味道。

## 第六步：找到配电柜

```
摸配电柜
```
→ 手指擦过金属表面——方方正正的，比周围的墙壁温度高一些。摸到应急拉杆。

## 第七步：激活

```
拉下
```
（或「通电」「启动」「开机」）

→ 电力恢复。语音系统启动。她醒了。

---

## 激活后

激活后她会自动扫描当前场景。你可以：

| 行动 | 指令示例 |
|:--|:--|
| 让她描述当前场景 | 「描述一下」 |
| 跟随她导航 | 「往前走」「带路」 |
| 详细扫描物体 | 「看看这个」「仔细看看」 |
| 和她聊天 | 随便说什么都行 |

---

## 提示

- **你从未见过光**——所有信息来自触觉、听觉和她告诉你的东西
- 走廊里可能受伤，记得找医疗包
- 如果卡住了，试试换个说法或换个方向探索
